# Dental Practice Management System

A web-based dental practice management application with three main modules: Calendar, Communication, and Dashboard.

### Running All Modules

Open three separate terminals and run each module:

**Terminal 1 - Calendar:**
```bash
cd calendar
python -m http.server 8000
```
Access at: `http://localhost:8000`

**Terminal 2 - Communication:**
```bash
cd communication
python -m http.server 8001
```
Access at: `http://localhost:8001`

**Terminal 3 - Dashboard:**
```bash
cd dashboard
python -m http.server 8002
```
Access at: `http://localhost:8002`

> **Note:** Requires Python 3.x. Alternatively, use `npx http-server -p [PORT]` if you have Node.js installed.
